# bashUtils

В данном проекте реализованы утилиты командной строки `cat` и `grep` на языке Си.

Реализованные функции полностью соответствуют оригинальным утилитам по:
- принципу работы,
- поддерживаемым флагам,
- поведению в различных сценариях использования.

Для проверки корректности реализации проводилось сравнение вывода написанных утилит с выводом системных команд `cat` и `grep` с помощью тестового Bash-скрипта.

Сборка и тестирование проекта осуществляются через Makefile. Доступны следующие цели:

- `all` — сборка всех утилит  
- `build` — альтернативная цель для сборки  
- `test` — запуск тестов  
- `_7214_cat` — сборка утилиты `cat`  
- `_7214_grep` — сборка утилиты `grep`  
- `clean` — удаление скомпилированных файлов
